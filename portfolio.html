1.1.9 Project

This is our first project of the term that we have had in PLTW 1.1.9. Here we have an ocean containing a curious turtle who is wandering to find its way in the sea. We made the turtle wander randomly. In the ocean we have created multiple surroundings such as a shark, kelp, and an octupus. When you first run the code you are asked a question about the oldest turtle in the world. If answered correctly, it will have one output whereas if answered incorrectly it will have a different output. We include multiple fun facts about the oldest turtle(Jonathan) that we thought were cool.
